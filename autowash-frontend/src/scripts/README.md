# Frontend Scripts

JavaScript duoc tach theo nhom:

- `api`: client goi backend Spring Boot, gom refresh token 401.
- `store`: session/state nhe dung cho demo frontend.
- `utils`: DOM va format helpers.
- `modules`: logic theo feature auth, customer, staff, admin.

Khi chuyen sang Java Servlet/JSP, co the giu nguyen cac module nay va chi thay duong dan HTML/template.
